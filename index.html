MiceBlock - A Minecraft Texture Pack

Mice Blocks is a Minecraft Resource Pack Inspired by oCd Pack, with a simple and consistent design, Mice blocks only use the Minecraft Vanilla color palette. We are at Beta stage of development, so be prepared for changes in the future. Currently, only Lite version is available for download.

How can I install this resource pack?


You can see a full tutorial in the Download area Here

Why Default version is not available for download?


Minecraft has a lot of textures. At moment, I only have textures for blocks and some elements of the interface, which will be available in the Lite version. items, creatures, weapons, tools and particles will be released later.

Wait! This is another copy of oCd Pack!


As previously stated, MiceBlocks is inspired by oCd pack, so there are similarities. it's important to highlight that every single texture in this pack was made from scratch.

Plans for the future?


A lot of plans! I don't want this texture pack to be another oCd low brain power copy. I want animated textures, overlays textures, connected textures, block variants, all that good stuff. Of course, the priority now is to finish the Default version.
