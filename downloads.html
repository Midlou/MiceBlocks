How to install


  1. Download the resource pack;
  2. Open the folder to which you downloaded the file;
  3. Copy the .zip file;
  4. Open Minecraft;
  5. Click on "Options", then "Resource Packs";
  6. Choose "Open resource pack folder";
  7. Paste the file into the resource folder in Minecraft;
  8. Go back to Minecraft, the resource pack should now be listed.

  • If not listed, go to the main menu by pressing Esc and repeat step 5).
  • You can see more information Here (Minecraft Wikipedia).
